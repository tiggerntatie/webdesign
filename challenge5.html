Summary

Unleash the Big Guns of CSS. You will learn the techniques for styling and positioning just the way you want!

Purpose

With the exception of form elements, you have written HTML using the most important and versatile tags available. In terms of style, you have been limited to colors and font styles, but very little to do with positioning. This challenge will require you to learn about position elements using float and clear and refine your styling using tag class attributes. You will also learn how to make a sexy navigation bar.

Due Date

This assignment is due on Friday, September 30. You will be able to work on it during the X.

Assignment

Right or wrong, many web sites have adopted similar patterns to their overall layout. The typical site has:

  • A header with a logo (typically on the left)
  • A horizontal navigation link bar or..
  • A vertical navigation link bar in the...
  • Sidebar column on the left hand side of the screen and...
  • Maybe even a sidebar column on the right hand side and...
  • The main content stuck in the middle somewhere.
  • A footer across the bottom with legal mumbojumbo and copyright, etc.

Web sites don't have to be styled like this, however. It is just a convention and conventions are begging to be messed with. Before you can become a renegade hipster web designer you really should master the fundamentals!

The Oatmeal wrote one of my favorite web comics about web design. Read the page at this link then make your own web page devoted to a type of food (tuna fish, anyone?). Here are the things you will have to do for the assignment:

  1. Create a new file on the server: challenge05.html - this exact name is important!
  2. Your page must be styled using an external CSS file named challenge05.css and customize it to suit the overall look you want.
  3. Your page must have a background image that enhances the style of the site, using the CSS background-image attribute of the body.
  4. Your page must have at least one <div> block for a header, styled using the float attribute.
  5. Your page must have at least two additional <div> blocks for content and one more for a footer using the float and clear attributes.
  6. Your page will have a list of at least 3 links (using the <ul> tag) that must be styled to become your navigation bar (vertical or horizontal).
  7. Add “Valid” icons for XHTML 1.1 and CSS to the bottom of your page.
  8. Your HTML and CSS should validate with no errors or warnings.

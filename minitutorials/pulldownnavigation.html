While it is true that most of the fancy pulldown, flyout super menus are designed using Javascript, it is possible to do a reasonably nice job using pure CSS! I have adapted a simple CSS-based menu design for this page.

Your menu links must be organized as a one or two-level hierarchy of unordered lists. For example, the menu at the top of this page is coded using the following HTML: 


<ul class="sitenavigation">
  <li><a href="#">One</a></li>
  <li><a href="#">Two</a></li>
  <li>
    <span>Three...</span>
    <ul>
      <li><a href="#">Sub One</a></li>
      <li><a href="#">Sub Two</a></li>
      <li><a href="#">Sub Three</a></li>
    </ul>
  </li>
  <li><a href="#">Four</a></li>
  <li><a href="#">Five</a></li>
</ul>

Notice that the links are all set to "#", which means they won't go anywhere. You should replace those with whatever HTML links you need. If you are working with Challenge 9, then the web page you are trying to style is already written using this method!

Here is the corresponding CSS file. I have tried to simplify this a little from the original author's version, and I have intentionally set colors and backgrounds that make the design ugly. You must start by making a complete copy of this CSS file and changing the text, border, background, etc., styles to suit your page design.

Don't worry if you don't understand all of this CSS right away. If you're curious, read the CSS and think about it line by line. I don't understand all of it either, but that doesn't stop me from borrowing it and modifying it!

How to use <table> tags

The <table> tag is how you organize any information that should be presented in the form of a table. For example:

This is a table caption
A Column Heading B Column Heading C Column Heading
A cell Another cell Yet another cell!
A cell in the next row And another And so on...

The HTML code for this looks like: 


<table>
  <caption>This is a table caption</caption>
  <tr>
    <th>A Column Heading</th> <th>B Column Heading</th> <th>C Column Heading</th>
  </tr>
  <tr>
    <td>A cell</td> <td>Another cell</td> <td>Yet another cell!</td>
  </tr>
  <tr>
    <td>A cell in the next row</td> <td>And another</td> <td>And so on...</td>
  </tr>
</table>

This example doesn't look like much: there are no borders around the table cells, the way you would expect to find in a word processor. We will spend more time making tables look pretty when we get to styling pages with CSS.

Every table is enclosed inside <table> and </table> tags. Within the table tags, define table rows using <tr> and </tr> tags. Within rows, define cells using <th> tag pairs for heading cells and <td> for ordinary data cells. Easy!

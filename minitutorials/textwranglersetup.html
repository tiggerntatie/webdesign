How to Set Up TextWrangler

  • Find the TextWrangler application and execute it.
  • Go to TextWrangler/Preferences/FTP Settings and add an SFTP bookmark for your account on netdenizen.org using the server sftp settings. The TextWrangler configuration does not have a place to enter the server Web URL.
  • When you need to create a new file on the server use File/New/SFTP/FTP Browser and select your new SFTP bookmark in the connect dialog. Click the New... button in the file browser to create a new folder or file.
  • When you want to open an existing file on the server, go to File/Open from FTP/SFTP Server and select the file you want.
  • When you want to save a file to the server, press command+s
  • There is no preview window in TextWrangler. You must view your changes as they happen on the web server, using a web browser.

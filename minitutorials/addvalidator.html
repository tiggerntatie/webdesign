How to add a validator link to your page

Since it is a pain in the neck to switch remember how to find the web page validator, it is convenient to add a special link on your web pages that will automagically link to the validator and validate your page. To add this link to your pages copy the following HTML code into the body of your page (between <p> tags, for example): 

<a href="http://validator.w3.org/check?uri=referer">
  <img src="http://www.w3.org/Icons/valid-xhtml11" alt="Valid XHTML 1.1" height="31" width="88" />
</a>

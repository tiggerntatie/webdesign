Styling the Background with CSS

Every displayable element on an HTML page has a background. By default, the background is simply white. Many interesting style effects are accomplished by either changing the background color, or by using an image for the background, with or without repetition. Often an image is combined with a background color in such a way that the edge of the image blends seamlessly with the color, giving the appearance of a much larger background image. The table below lists some of the things you can dow with backgrounds.

Don't forget to experiment with changing the background style of a variety of elements. And don't forget to examine the HTML and CSS of web pages you admire!

CSS Attributes For Backgrounds

attribute example value used for...
background #eee url(background.png) repeat-x scroll; Specify a combination of background attributes at one time: color, background image, repetition and attachment. These qualities can also be set individually (see below).
background-color #353; Specify the background color using RGB code like #353 (short) or #3355ff (long).
background-image url(background.png); Specify the name of an image to use as the background.
background-repeat no-repeat; Images can be repeated vertically and horizontally (repeat), just vertically (repeat-y), horizontally (repeat-x), or not at all (no-repeat).
background-attachment scroll; Images in the background can scroll with the rest of the content (scroll), or be fixed in the browser window (fixed).
background-position top right; Placement of the background image can be aligned at the top, right, bottom, left, center or a combination of these. You can also specify distance from the left and top margin.

There are a lot of interesting effects that you can achieve with background styling. See the Opera tutorial on background styling for more ideas.

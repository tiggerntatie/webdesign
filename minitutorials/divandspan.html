<div> and <span>

The elements (or tags) that you use in HTML come in two flavors: block and inline. Elements like <p>, <ul> and <h1> are block elements. Block elements always force an end-of-line in any text that precede them, and force a new line to start after the element.

The inline elements are, for example, <strong>, <em> and <img>. When these elements appear in your HTML they do not interrupt the flow of the line of text. They do not force a new line on your page.

If you want to group parts of your page in an arbitrary way, either as a block or as an inline group of text, you can use the <div> and <span> elements. The <div> element is used to create a collection of blocks that behaves like a single block element and <span> is used to create a collection of inline text that behaves like an inline element.

For example, I grouped the two preceding paragraphs inside a pair of <div> elements and styled it in CSS.

The CSS looks like this (and view the page source to see what the HTML looks like: 


div.box {
  background-color: #eee;
  padding: 3em;
  width: 80%;
}

Styling Navigation Links with CSS

Most web sites or pages have a list of important links (<a> tags) that help the reader find other pages on the site. Since it is merely a list of links, the preferred method of making this navigation list is with an HTML list such as <ul>. A navigation list like this might look something like: 


<ul>
  <li><a href="./index.html">Web Design Home</a></li>
  <li><a href="./calendar.html">Assignment Calendar</a></li>
  <li><a href="./studentwork">Student Work</a></li>
</ul>

Which looks like this:

  • Web Design Home
  • Assignment Calendar
  • Student Work

This gets the job done, but it looks awful. That is where CSS comes to the rescue!

CSS Attributes For Styling Lists of Links

Instead of coding your links like the previous example, add a class attribute like this: 


<ul class='navigation'>
  <li><a href="./index.html">Web Design Home</a></li>
  <li><a href="./calendar.html">Assignment Calendar</a></li>
  <li><a href="./studentwork">Student Work</a></li>
</ul>

Then the following CSS attributes could used to style only the links in a list that is marked with the navigation class: 


ul.navigation { 
  margin: 0;
  padding: 0.5em;                     /* a little space around the menu                               */
}

ul.navigation li {
  display: inline;                    /* turns a <li> into an inline tag, instead of a block     */
                                      /* this also gets rid of the bullets                             */
  background-color: #eee;             /* set it apart visually                                         */
  padding: 0.1em 0.3em 0.1em 0.3em;   /* give it some added bulk and size                              */
                                      /* these sizes are for top right bottom and left, respectively   */
}

ul.navigation a {
  text-decoration: none;              /* no underline for these links                                  */
  font-family: verdana, sans-serif;   /* a different font to set them apart from text                  */
}

ul.navigation li:hover {
  background-color: #ccc;             /* change the link background color when the user hovers         */
}

And the end result is this:

  • Web Design Home
  • Assignment Calendar
  • Student Work

Let's talk about how this CSS works. Instead of just referring to the <ul> tag in each CSS block, it refers to ul.navigation. This means that the styling rule doesn't apply to just any <ul> tag: it refers to only the <ul> tags that have their class attribute set to navigation. Furthermore, when two tags are listed one after the other with a space between them (ex. ul.navigation li) it means that you are styling every <li> tag that appears inside a <ul> tag with the navigation class.

Go back up to the second HTML example and look at the snippet of code: The <ul class='navigation'> that begins the unordered list has an additional attribute, class, set equal to navigation. Now only this unordered list will have the special styling. Any ordinary list will just look like a simple list of bullet points.

Notice that to use the class attribute in your CSS, you have to refer to the tag, followed by a period, then the name of the class that was set in the HTML. Pro-tip: don't name your classes in a way that suggests the style (that is, don't name a class 'bold' or 'big-font'). Your class names should reflect the semantics of the content: it should suggest how this piece of the page is meaningfully different from everything else. In the example on this page I have used the class named navigation, which indicates the purpose of the list, but nothing about its appearance.

Challenge 2: Create a Recipe Page

Summary

Use HTML (HyperText Markup Language) to write a web page of your favorite recipe.

Purpose

This assignment gives you practice writing your first web page by hand. There are many tools and web sites (e.g. Dreamweaver or Google Sites) that take care of HTML details for you, but a good web designer (like you!) understands the language of web pages (HTML) and knows how to write web pages “by hand”.

Assignment

What is your favorite recipe? This could be a recipe for pancakes, or a recipe for something entirely different! How would you write a detailed “recipe for disaster?” Your assignment is to create a new web page using your personal hosting account on netdenizen.org. The page must be your favorite recipe for something.

Here are the things you have to do for the assignment:

  1. Create a new file on your netdenizen.org site. Name it challenge02.html.
  2. Write your recipe in the file using Dreamweaver in Code mode (Code button is highlighted at the top of the editor window).
  3. Your recipe HTML should have examples of the HTML elements we have looked at this week:
    • <title>
    • <h1>, <h2>, etc. (at least one)
    • <p>
    • <ul> and <li> (for a list of ingredients)
    • <ol> and <li> (for a numbered list of steps)
    • <dl> and <dt>, <dd>(for a list of defined terms)
  4. Your page should validate with no errors or warnings.
  5. Using your school e-mail account, send me an e-mail with a link to your new web page.

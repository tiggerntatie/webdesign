Assignment

The web didn't always depend on the two pillars of web design: HTML and CSS. Once upon a time everything was written using HTML alone! Style information was embedded in the HTML elements themselves, and changing the style or appearance of a web page was very inconvenient once it had been completely written.

In the early 2000s a web site appeared that demonstrated the promising marriage of CSS and HTML like nothing else before: CSS Zen Garden. The Zen Garden was a single page web site that allowed the visitor to select from a variety of user-contributed CSS files. Although each CSS file made the web page look completely different, the underlying HTML file was never altered.

CSS Zen Garden's mission has been accomplished and today it no longer accepts submissions of new CSS files. In Challenge 9 you will borrow the CSS Zen Garden concept and create a CSS file only that will style the Hanover High School home page.

For this challenge I have recreated the HHS home page as a simple HTML page with numerous <div> blocks, section titles and all of the original navigation links. I have left out all images that do not semantically enrich the site. I have removed all styling information from the page. Your new HHS homepage consists of 9578 characters, compared with the original HHS page that has 26378 characters.

I have placed a link to the new HHS homepage HTML in your public folder. You can not edit this file. It also pulls in a file called challenge09.css which does not exist in your public folder. You must create this CSS file and edit it to make an awesome looking HHS home page!

  1. Create a new file on the server: challenge09.css - this exact name is important!
  2. Your CSS must validate with no errors or warnings. I have already taken care of the HTML validation!
  3. There is no third requirement! I will not limit your imagination by giving you a laundry list of tasks to perform. You have until Friday to do the best possible job!

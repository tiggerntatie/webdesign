Summary

Using the project rubric, how are you doing?

Purpose

In addition to the usual weekly plans and status reports, every project needs to have a sense of where it stands. At this stage in the semester you should have a project that has made substantial progress towards completion. It is time to formally assess the progress of the site by comparing it against the rubric that will be used to grade the finished project.

Due Date

This self assessment is due on Monday, December 19.

Assignment

With your partner, if you have one, read the project rubric and create a new web page in your plan folder that describes how you interpreted the rubric in your specific case. In other words, what would your grade be today if you stopped working on the project?

Here is what you need to do:

  • Create a file named selfevaluation01.html in your plan folder. The name and location are important! If this is your second self evaluation then it would be named selfevaluation02.html. Get it?
  • The page title should indicate your project name and which self evaluation this is (1, 2, etc.).
  • Include a simple unordered list (<ul>) that lists each graded item on the project rubric and how many points you would receive for your project at this time.
  • Your self evaluation must validate with no errors or warnings (of course!).

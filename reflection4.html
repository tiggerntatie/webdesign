Web Design Reflection 4: Your Wish List?

Last week I asked you to think about your ideas for a final project web site to work on during the second quarter. This week I would like you to think about what you need to make your ideas come to life. Specifically, as you think back on what you have learned so far, do you think you have picked up the skills and knowledge that you (and your partner) will need to execute your ideas?

I will use this reflection's results to help guide the course curriculum for the remainder of the second quarter.

  1. Create a web page in your public folder and name it reflection04.html.
  2. Please answer the following questions in your page:
    • What is the most important feature of your (planned) site that you have no idea how to create? Describe the feature with thorough detail!
    • Find an existing web site (or sites) that also have this feature, or one similar to it. Include a link to this site in your reflection. Describe the feature on this other site so that I will be able to find it.
    • If you can identify the specific skill or knowledge that you lack for implementing this feature, please tell me about it.
    • Inspect the source of the existing web site. Can you find the section of HTML that is important for this feature?
    • If you would like to, repeat these steps for any additional features that you think will present a problem.
  3. Your reflection web page should validate (of course!).

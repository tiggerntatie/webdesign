Web Design Reflection 3: Think About A Project

Now that you have some experience writing web pages and uploading files to a web server, you should know that you have developed core skills needed to be a professional web developer or designer. We have several more important topics to explore, but in the mean time this is a good time to start thinking about where you want to go during the second half of the course.

During the second quarter, you (and your partner) will design and create a full web site devoted to something that you are both interested in. For this reflection you will:

  1. Create a web page in your public folder and name it reflection03.html.
  2. In your page, present an idea for a web site that you would like to develop. You should include:
    • A possible name for the site.
    • Will it be a site with information? Selling something? Promoting a cause? What?
    • Are there other sites like it out there? What would set yours apart?
    • Describe the look and feel of the site. Is there another site you would like to emulate?
    • Discuss your idea with a potential partner. What is his/her reaction?
  3. Your reflection web page should validate (of course!).

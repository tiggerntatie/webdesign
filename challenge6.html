Summary

It's time to make your web pages more interactive! Write web pages that ask for user input and react in some way.

Purpose

In the last challenge you learned to give your pages a new degree of dynamic behavior by styling your navigation links with CSS. In this challenge you will write HTML that asks your reader for information using the <form> and <input> elements, then responds by changing the appearance of the page itself using Javascript. You will also practice your skills with navigation menu styling and arranging blocks using <float> and <clear>.

Due Date

This assignment is due on Friday, October 7.

Assignment

In this assignment you will learn to create a form, style it with CSS and make it interact with the web page using Javascript. By way of explanation, this page includes an example for you to inspect:

Visitor Information

Hello, my name is ? and I was born in ? in the month of ? during the ?. I have a secret, which is: ? and my political tendency is rather ?. What is more, I have a unique world view which says: ?.

Your assignment is to create a web page that works similar to one of these Wacky Web Tales. Unlike the Wacky Web Tales pages, however, your page will not display the finished story on a different page. You should use this assignment as an example and modify it as needed to become your own version of a Wacky Web Tale.

This challenge will be, well.. challenging. Be sure to read the mini tutorials carefully and look at the source of this assignment for examples.

  1. Create a new file on the server: challenge06.html - this exact name is important!
  2. Your page must be styled using an external CSS file named challenge06.css, customized to suit the overall look you want. If you want to base your new CSS file on the one you wrote for challenge 5, I recommend opening the old CSS file then saving it back to the server using SFTP and renaming it to challenge06.css. Do not try to use the same CSS file for all your assignments because it tends to break the styling of your old pages.
  3. Your page must include a list of web form elements that will be used to generate words for your story. Inspect the source of this assignment carefully and use it as an example. Your form should include at least:
    • Six <input> elements with type='text'.
    • Three <input> elements with type='radio'.
    • Three <input> elements with type='checkbox'.
    • One <input> element with type='password'.
    • One <select> element with at least six <option> elements inside it.
    • One <textarea> element.
  4. Your page must link to an external Javascript file named challenge06.js. You should copy and modify the javascript file that is used in this assignment! In particular, you should modify the block of code that looks like this: 
    
    this.addWordbyID("first-name", "first-name-dest");
    this.addWordbyID("birthplace", "birthplace-dest");
    this.addWordbyID("birth-month", "birth-month-dest");
    this.addRadioButtonbyName("birth-decade", "birth-decade-dest");
    this.addWordbyID("secret", "secret-dest");
    this.addCheckBoxbyName("leaning", "leaning-dest");
    this.addWordbyID("world-view", "world-view-dest");
  5. Your page must have a background image that enhances the style of the site, using the CSS background-image attribute of the body or some other element.
  6. Your page must have at least two elements (such as <div>) styled using the float and/or clear attributes.
  7. Your page will have a list of at least 3 links (using the <ul> tag) that must be styled to become your navigation bar (vertical or horizontal).
  8. Add “Valid” icons for XHTML 1.1 and CSS to the bottom of your page.
  9. Your HTML and CSS should validate with no errors or warnings.

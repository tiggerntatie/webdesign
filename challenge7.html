A few more things to play with

This challenge just introduces a handful of other goodies you can use to style your page backgrounds.

The page you are looking at has several different things going on in the background. First of all, the background-color has been set to that salmon like color using CSS: 


background-color: #ffb95f;

There is a nifty gradient effect going on at the top of the page. This is done by repeating a single, small gradient image across the top of the page. You can take a look at the image here. The (incredibly ugly) logo image is a separate image that is overlaid on the gradient. Both of these images are specified in the CSS like this: 


body {
  background-image: url(logo7.png), url(gradient7.png);
  background-position: 20px 40px, top center;
  background-repeat: no-repeat, repeat-x;
  background-color: #ffb95f;
  font-family: verdana;
}

Notice that the background color was carefully chosen to match the very bottom edge of the gradient image, so that they would blend seamlessly together.

Alpha blending

Do you notice how the background of the h1 heading and body text appears to be semi-transparent? This is an effect available with CSS that is supported in most of the recent web browsers. To do this, you have to specify the block (using div in this case) background-color using a slightly different method. For example: 


  background-color: rgba(0,0,0,0.1);

This specifies a black background (the 0,0,0 are decimal equivalents of the hex numbers that are usually used to specify colors. These can range from 0 to 255. So, for example, you would specify the color white as rgb(255,255,255)). The rgba command adds an alpha channel, which is a fancy term for transparency. The last number in the rgba command is a decimal value between 0.0 and 1.0, which specifies how transparent the color is. An alpha value of 0.0 is perfectly transparent, and 1.0 is perfectly opaque.

How to make the gradient and logo

All of the images for this challenge page were created using an online image editor called Sumo Paint.

The gradient image

  • In Sumo Paint, create a new image.
  • Set the image size to 10 pixels by 100 pixels.
  • Click the gradient fill tool on the toolbar (left).
  • Select one of predefined sexy gradients.
  • Click-drag from the top of your image to the bottom to fill the background.
  • When you have one you like, save it to your computer as a png file.
  • Upload the image to our web server using Cyberduck.

The background color

  • With the gradient image still in Sumo Paint...
  • Select the eyedropper tool.
  • Click at the very bottom of your gradient image.
  • Look to the color palette (on the right). The color you have "picked" using the tool should appear in hex form (e.g. #ffb95f). Make a note of this.
  • Use the color from the previou step as the background color for your page.

The ugly logo

  • In Sumo Paint, create a new image and size it for 150px by 150px (for example).
  • Create a new layer (lower right)
  • Delete the original layer with its white background
  • Create the most amazing logo you can imagine in this 150x150 pixel canvas.
  • Save your image to your computer as a png file.
  • Upload the image to our web server using Cyberduck.

Bring it all together

Now when you specify the background information, you can list both the gradient and logo images together. The first image listed will appear on top of the others which can give you a nice layered effect. Separate images using commas like this: 


background-image: url(logo7.png), url(gradient7.png);
background-position: 20px 40px, top center;
background-repeat: no-repeat, repeat-x;
